Hack the North 2024: Fangler

This is an Arduino Hardware project that I was able to work on with a group of fellow high school students. It is a fitness device that regulates body temperature based on weather and provides proper lighting at night. Link to Devpost Submission

Arduino Documentary #1

This is the first video in a series I am starting where I learn how to use the Arduino, I hope to learn as much as I can about circuitry and electronics.

Lava Escape

This is the first game that I have fully completed and tested. It uses the Java language to create a game where the user is a block trying to escape the rising lava in a volcano, using the falling ash blocks to jump up. It is a score based system, meaning that the goal is to survive as long as possible!

Making an Arcade Machine

This is a project that I have worked on during the summer of 2024, it is my take on a bartop arcade machine made of MDF wood panels that I have cut into shape. It utilizes a used monitor and Raspberry Pi 4 2GB as the screen and computer.

Making an Arcade Machine Pt.2

This is the second part to my 2 part series in which I built my very own arcade machine. It utilizes a used monitor and Raspberry Pi 4 2GB as the screen and computer.

CREATE Design Challenge: BeatTrends Demo

This is a Project where I and a group of students used Python data visualization/manipulation libraries to create graphs based on music data collected between the early 90s to 2020. We hoped to bring less popular genres of music to the mainstream media where the type of music is largely dominated by pop.
